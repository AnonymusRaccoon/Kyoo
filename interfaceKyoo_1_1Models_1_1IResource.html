◆ Slug

string Kyoo.Models.IResource.Slug
get

A human-readable identifier that can be used instead of an ID. A slug must be unique for a type of resource but it can be changed.

There is no setter for a slug since it can be computed from other fields. For example, a season slug is {ShowSlug}-s{SeasonNumber}.

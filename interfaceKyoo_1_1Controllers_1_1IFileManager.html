◆ FileResult()

IActionResult Kyoo.Controllers.IFileManager.FileResult ( [CanBeNull] string  path,
bool  rangeSupport = false,
string  type = null 
)

Used for http queries returning a file. This should be used to return local files or proxy them from a distant server

If no file exists at the given path or if the path is null, a NotFoundResult is returned to handle it gracefully.

Parameters
pathThe path of the file.
rangeSupportShould the file be downloaded at once or is the client allowed to request only part of the file
typeYou can manually specify the content type of your file. For example you can force a file to be returned as plain text using text/plain. If the type is not specified, it will be deduced automatically (from the extension or by sniffing the file).
Returns
An IActionResult representing the file returned.

Implemented in Kyoo.Controllers.FileManager.

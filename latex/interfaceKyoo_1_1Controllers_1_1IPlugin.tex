\doxysubsection{Detailed Description}
A common interface used to discord plugins 

You can inject services in the \mbox{\hyperlink{interfaceKyoo_1_1Controllers_1_1IPlugin}{I\+Plugin}} constructor. You should only inject well known services like an I\+Logger, I\+Configuration or I\+Web\+Host\+Environment.

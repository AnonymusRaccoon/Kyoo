Hi there! We\textquotesingle{}re thrilled that you\textquotesingle{}d like to contribute to this project. Your help is essential for keeping it great.

Contributions to this project are \href{https://docs.github.com/en/github/site-policy/github-terms-of-service\#6-contributions-under-repository-license}{\texttt{ released}} to the public under the \href{https://github.com/AnonymusRaccoon/Kyoo/blob/master/LICENSE}{\texttt{ G\+P\+L-\/3.\+0 License}}.\hypertarget{md_CONTRIBUTING_autotoc_md1}{}\doxysection{Submitting a pull request}\label{md_CONTRIBUTING_autotoc_md1}

\begin{DoxyEnumerate}
\item \href{https://github.com/AnonymusRaccoon/Kyoo/fork}{\texttt{ Fork}} and clone the repository
\item Install the toolstack and/or download precompiled binaries for the transcoder (see \href{https://github.com/AnonymusRaccoon/Kyoo\#development--build}{\texttt{ Development \& Build}})
\item Create a new branch\+: {\ttfamily git checkout -\/b my-\/branch-\/name}
\item Make your changes (Please try to follow the same coding style as what has already been made -\/ indent with tabs, align with spaces etc)
\item Make sure the tests pass\+: {\ttfamily dotnet tests}
\item Push to your fork and \href{https://github.com/AnonymusRaccoon/Kyoo/compare}{\texttt{ submit a pull request}}
\item Pat your self on the back and wait for your pull request to be reviewed and merged.
\end{DoxyEnumerate}

Here are a few things you can do that will increase the likelihood of your pull request being accepted\+:


\begin{DoxyItemize}
\item Write tests.
\item Make sure the {\ttfamily \mbox{\hyperlink{README_8md}{R\+E\+A\+D\+M\+E.\+md}}} and any other relevant {\bfseries{documentation are kept up-\/to-\/date}}.
\item We try to follow \href{https://semver.org/}{\texttt{ Sem\+Ver v2.\+0.\+0}}. Randomly breaking public A\+P\+Is is not an option.
\item Keep your change as focused as possible. If there are multiple changes you would like to make that are not dependent upon each other, consider submitting them as {\bfseries{separate pull requests}}.
\item Write a \href{http://tbaggery.com/2008/04/19/a-note-about-git-commit-messages.html}{\texttt{ good commit message}}.
